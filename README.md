
# Hyperaudio Lite Editor
A lightweight transcript editor for editing and correcting machine generated transcripts.

Super Lightweight :snowflake: 
Lightning Fast :zap: 
Accessible :couple_with_heart: 
Installable :computer: 
Portable :iphone: 

### Formats

Export your transcript as text, interactive transcript, captions and subtitles.

### How to Use

Edit in the usual way â€“ add punctuation, create and merge paragraphs etc. Place speaker names in square brackets ie [Maria]. It's recommended that in order to maintain word timings you don't paste large blocks of text.  Double-click on words to set the playhead at that point.
